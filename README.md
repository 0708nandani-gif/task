# Digital Clock & Stopwatch

A frontend project built with HTML, CSS, and JavaScript featuring:

- **Live Digital Clock** — displays current time, date, and timezone, updating every second
- **Stopwatch** — start, stop, lap, and reset functionality with a visual progress ring
